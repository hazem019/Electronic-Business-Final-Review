Student Data Analysis

You have student data and want to perform descriptive and inferential analysis using SPSS. Follow each task step-by-step below.

Task 1
Descriptive Analysis
Extract Mean, Minimum, Maximum, Range, and Std. Deviation for students' grades.
SPSS Steps
1 File ➔ Import Data ➔ Excel ➔ Excel Name ➔ File Type ➔ Open
2 Analyze ➔ Descriptive Statistics ➔ Descriptives
3 Select Variable: Grade
4 Options ➔ Check: Mean  ·  Minimum  ·  Maximum  ·  Range  ·  Std. Deviation
5 Click OK

Task 2
ANOVA Test
Determine whether there is a significant difference in grades between departments A and B.
SPSS Steps
1 File ➔ Import Data ➔ Excel ➔ Excel Name ➔ File Type ➔ Open
2 Analyze ➔ Compare Means ➔ One-Way ANOVA
3 Select Variable: Grade
4 Click OK
🔗
Task 3
Spearman Correlation Test
Determine the relationship between Hours Study and Grade using Spearman correlation.
SPSS Steps
1 File ➔ Import Data ➔ Excel ➔ Excel Name ➔ File Type ➔ Open
2 Analyze ➔ Correlate ➔ Bivariate
3 Select Variables: Hours Study & Grade
4 Select Correlation Coefficient: Spearman
5 Click OK
